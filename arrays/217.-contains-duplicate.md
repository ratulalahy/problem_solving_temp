# 217. Contains Duplicate

```python
class Solution:
    def containsDuplicate(self, nums: List[int]) -> bool:
        dict_exists = {}
        for cur_num in nums:
            if cur_num in dict_exists:
                return True
            else:
                dict_exists[cur_num] = True
        return False
        

```
