```python
class Solution:
    def heightChecker(self, heights: List[int]) -> int:
        h_sorted = sorted(heights)
        
        count_msmatch = 0
        for i in range(len(heights)):
            if heights[i] != h_sorted[i]:
                count_msmatch += 1
        return count_msmatch
```
