# 88. Merge Sorted Array

```python
class Solution:
    def merge(self, nums1: List[int], m: int, nums2: List[int], n: int) -> None:
        """
        Do not return anything, modify nums1 in-place instead.
        """
        ptr_in = (m+n)-1
        n -= 1
        m  -= 1
        while m >= 0 and n >= 0:
            if nums2[n] > nums1[m] :
                nums1[ptr_in] = nums2[n]
                n -= 1
            else:
                nums1[ptr_in] = nums1[m]
                m -= 1
            ptr_in -= 1
            
        while n >= 0:
            nums1[ptr_in] = nums2[n]
            ptr_in -= 1
            n -= 1
```
